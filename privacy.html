Privacy Policy

Last updated: June 2025

This Privacy Policy explains how the Discord bot ("the Bot") collects and uses data.

Data Collected

The Bot stores configuration data such as role IDs, channel IDs, and team assignments. This data is saved in server-local configuration files and not shared with any third parties.

Usage Logs

The Bot may store usage statistics (e.g. how many screenshots are approved per team). This is only used for event tracking and is not shared externally.

Security

We take reasonable steps to secure stored configuration data, but we cannot guarantee absolute security.

Contact

If you have questions, contact the server administrators or the bot owner directly via Discord.
